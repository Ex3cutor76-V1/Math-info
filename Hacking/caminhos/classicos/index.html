Hackers clássicos:

Esses são aqueles hackers clássicos que você se torna, a maioria se torna um deles no começo após iniciar.

Hacker: O mais clássico de todos

Aqui é bem simples até, são as pessoas que exploram os sistemas, redes ou até softwares para aprender, testar ou até melhorar algumas coisa, a intenção do hacker pode ser tanto ética quanto anti-ética, e não, nem todo hacker é criminoso.

Cracker: O "hacker ilegal"

Esse já é aquele que usa habilidades técnicas para quebrar sistemas, softwares ou proteções digitais de forma ilegal, esse sim você pode acusar de criminoso.

Script Kiddie: Iniciante

Eu já citei script kiddie tantas vezes que meu Deus, mas blz, o script kiddie resumidamente é só os iniciantes que usam ferramentas já prontas, sem ao menos entender como elas funcionam de forma técnica.

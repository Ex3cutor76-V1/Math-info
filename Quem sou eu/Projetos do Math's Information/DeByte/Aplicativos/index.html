Aplicativos do DeByte:

A maioria dos aplicativos instalados no DeByte são em CLI, poucos serão gráficos, afinal, a filosofia do DeByte é CLI-first o que explica a maioria das ferramentas serem em CLI (No caso em texto puro), porque além de se encaixarem na filosofia do DeByte, eles são extremamente leves, no qual é o objetivo do DeByte, que seria ser um sistema muito leve para computadores antigos.

Navegadores:

firefox-esr: Navegador principal/padrão do DeByte.

Tor browser: Navegador secundário, não como padrão, mas, como uma alternativa de anonimato.

Segurança:

ufw: O firewall do DeByte, garantindo para o user uma segurança básica (E não o DeByte não configura firewall para você, ele deixa o firewall desativado por padrão e deixa você configurar.)

GNUpg: Software para criptografar arquivos.

KeePassXC: Gerenciador de senhas offline o user guardar as senhas.

torsocks: Software que força aplicativos a usar a rede tor (Uso opcional).

xterm: Terminal de recuperação caso você quebre seu sistema.

Trabalho:

libreoffice: O pacote ofice gratuito.

Sistema:

htop: Gerenciador de tarefas.

xfe: Gerenciador de arquivos.

i3: Window Manager principal do sistema (Ele sustenta o sistema inteiro sozinho).

i3status: Status bar do sistema (No caso esse é o software que vai mostrar: Bateria, HD, RAM, Data e hora e até se você está conectado na internet).

rofi: Launcher de menu do sistema (Sem ele você não iria ver a lista de aplicativos do sistema)

terminator: Terminal principal do DeByte (É nele mesmo que o sistema irá iniciar primeiro).

bash: SHELL principal do sistema (Por favor, não quebrar).

apt: Gerenciador de pacotes (No caso o mesmo que o Debian ou Ubuntu).

Utilitários:

cmus: Player de música em texto.

irssi: IRC em texto.

mutt: Cliente de email em texto (Você só vai ter que configurar para colocar seu email).

bc: Calculadora em texto.

nano: Editor de texto.

network-manager: Esse é o software que vai te permitir conectar no wifi.

zram-tools: É o software que irá compactar sua RAM, assim te dando mais desempenho e não gastar tanta RAM (Exemplo: Se caso você tem 4GB de RAM e usa um software pesado ele vai pegar quase tudo da RAM, com o zram ele meio que vai te permitir ter +2GB RAM extras, porque ele vai comprimir o processamento de RAM.)

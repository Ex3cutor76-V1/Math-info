Atalhos do DeByte:

Uma curiosidade do DeByte? Você dificilmente usa mouse, o motivo? É porque o sistema inteiro você é obrigado a usar teclado, tipo não é a toa que o DeByte usa atalhos que bem esses atalhos são feitos pelos teclados e por aqui irei mostrar a você todos os atalhos do DeByte:

Por que o DeByte usa mais teclado do que mouse?:

Como o software que carrega o sistema inteiro nas costas é uma WM (Window Manager) chamada i3 ele simplesmente só aceita o teclado para poder mexer no sistema e isso explica porque no DeByte você usa mais teclado do que mouse.

Atalhos básicos:

Aviso: "mod" é na verdade a tecla Windows que também é conhecida como tecla "super".

mod + Shift + E --> Desliga o computador.

mod + Shift + R --> Reseta o computador.

mod + Enter --> Abre o Terminator (Ou o terminal).

mod + D --> Abre o launcher rofi.

mod + Shift + Q --> Fecha aplicativo.

Atalhos para se mover no sistema:

O DeByte mesmo com o i3 ele foi feito para possuir apenas 5 janelas, então... Sim você só terá 5 janelas para uso separado.

mod + 1 --> Move para a janela 1

mod + 2 --> Move para a janela 2

mod + 3 --> Move para a janela 3

mod + 4 --> Move para a janela 4

mod + 5 --> Move para a janela 5

Mover aplicativos para janelas específicas:

mod + Shift + 1 --> Move o aplicativo para a janela 1

mod + Shift + 2 --> Move o aplicativo para a janela 2

mod + Shift + 3 --> Move o aplicativo para a janela 3

mod + Shift + 4 --> Move o aplicativo para a janela 4

mod + Shift + 5 --> Move o aplicativo para a janela 5

Atalhos úteis:

mod + E --> Abre o xfe.

mod + H --> Move aplicativo para o lado esquerdo da tela.

mod + J --> Move o aplicativo para o lado de baixo da tela.

mod + K --> Move o aplicativo para o lado de cima da tela.

mod + L --> Move o aplicatvo para o lado direito da tela.

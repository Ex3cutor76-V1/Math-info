Criptografia:

Olá caro visitante, aqui no Math's Information já falamos sobre criptografia, só que falando de uma ferramenta em si, chamada de gpg, que claro não é só ela que realmente faz essas coisas, ainda existem outras ferramentas de criptografar arquivos, entretanto, não vamos falar sobre isso e sim vamos entender o que é uma criptografia.

O que é criptografia?:

É basicamente o processo de codificação de informações para proteger de acessos não autorizados, isso em si é bem importante, uma vez que se caso uma informação é confidencial, algo externo, pode ler a informação.

Como funciona uma criptografia?:

Para resumir, a criptografia pega os dados legíveis, no caso a mensagem e transforma em algo ilegível, tipo uma cifra assim, quem tem a chave certa consegue ler, assim, mesmo que alguém consiga os dados não irá conseguir entender os dados sem a chave correta.

Tipos de criptografia:

Criptografia Simétrica:

A chave para criptografar é a mesma para descriptografar os dados, é uma das mais rápidas, porque é apenas uma chave para descriptografar os dados, entretanto, é mais arriscado considerando que se alguém externo interceptar a chave, já que as chaves são as mesmas (Exemplo de Criptografia Simétrica: AES ou Advanced Encryption Standard).

Criptografia Assimétrica:

Nessa criptografia se utiliza 2 chaves diferentes, uma chave é pública (A chave pública é para criptografar) e a chave privada (Que é a chave que vai descriptografar), é uma das mais seguras, porque mesmo se a chave pública for interceptada, a chave privada ainda vai ser necessária para poder descriptografar os dados, entretanto, é mais lenta, pois é, segurança e privacidade tem custo (Exemplo de criptografia Assimétrica: RSA).

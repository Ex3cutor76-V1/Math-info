Como ver logs no Linux:

Certo certo, aqui no site falamos muito de segurança de fato... Mas como falar de segurança sem citar os logs? Afinal... Os logs são os registros do sistema.

Afinal, o que de fato é log?:

Antes de iniciar um comando que mostra as logs do sistema, é preciso entender primeiro o que é uma log, e bem em resumo é só o registro do sistema, no caso é o log que registra as coisas que você fez no seu computador como: horário de logou, horário que desligou, qual app ou até software que você executou e inclusive ele registra erros no sistema também.

journalctl:

O journalctl é um dos comandos que vai te dar acesso aos logs e de forma bem profunda e aqui está alguns comandos bem básicos:

journalctl: Com este comando você vê todos os logs, literalmente, todos os logs.

journalctl -e: Com este comando ele irá mostrar logs mais recentes.

journalctl -f: Mostra os logs em tempo real.

journalctl -u nome-do-serviço: Filtra os tipos de logs por nome dos serviços.

journalctl -p palavra: Basicamente filtra os logs por alguma palavra.

Sobre o "journalctl -p" eu recomendo honestamente que nas palavras use:

emerg
alert
crit
err
warning
info
debug

Porque sinceramente, fica mais fácil identificar erros e corrigir.

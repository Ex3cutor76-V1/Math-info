Arch Linux:

Como não falar do maior medo entre os nerds? Você provavelmente já viu aquele meme "I use Arch btw", pois é, eu lhe apresento o Arch Linux o sistema operacional mais "difícil" de instalar e uma das distros Linux mais difíceis de usar, ainda mais porque o Arch Linux é conhecido pelo seu minimalismo, então tipo quando você instala o Arch Linux pela primeira vez, você monta o sistema inteiro, mas, uma dica? Se não quiser passar sufoco escrevendo comando use o comando: archinstall que ele abrirá um menu "gráfico" em texto para você selecionar o que você quer, e só pra avisar, que caso faça isso saiba que é você que escolhe a configuração, ele não faz isso por você.

Informações principais:

Gerenciador de pacotes: Pacman

Filosofia: KISS (Keep It Simple, Stupid)

Modelo: Rolling Release (No caso atualiza continuamente)

Dificuldade de uso: Alto

Vantagens:

1) Sistema leve e rápido e sem "blotware" (no caso não vem aplicativo pré-instalado)

2) Sempre atualizado

3) Wiki Arch (Ele facilita para você mexer no sistema)

Desvantagens:

1) Não é amigável para iniciantes, qualquer erro de configuração você quebra o sistema.

2) Atualizações contínuas podem quebrar o sistema devido a compatibilidade.

3) Suporte comercial limitado (No caso é mais voltado para a comunidade do Arch Linux)

Site oficial do Arch Linux: https://archlinux.org/

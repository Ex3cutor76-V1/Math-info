Gentoo:

Agora sim chegamos em um ponto onde você enlouquece, porque essa distro é um inferno para instalar e usar uma vez que é terminal puro e ainda por cima você que compila cada software que instala, sabe o firefox? Exatamente, você compila ele.

Informações principais:

Gerenciador de pacotes: Portage (No caso ele compila softwares diretamente do código fonte)

Filosofia: Build It Yourself ("Construa você mesmo")

Modelo: Rolling release (Igual ao Arch só que mais flexível ainda)

Dificuldade: Extremamente alta.

Vantagens:

1) É extramemente otimizado e ŕapido.

2) Você possui total controle sobre o software e dependências via USE flags.

3) Atualização contínua com softwares sempre recentes.

4) Ótimo para aprendizado avançado de Linux.

Desvantagens:

1) Curva de aprendizado alta.

2) Compilar software's pode levar muito tempo dependendo do seu hardware (Reza pra sua CPU não ir pro 100%)

3) Atualizações podem quebrar o sistema se não for gerenciada com cuidado.

4) Suporte mais limitado para iniciantes.

Site oficial do Gentoo: https://www.gentoo.org/

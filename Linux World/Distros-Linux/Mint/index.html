Linux Mint:

Ah o doce Mint... Você provavelmente já viu ele no computador da sua vó ou de algum curso gratuito que você já fez e pensou que era Windows e bem o Linux Mint é uma distro Linux baseada em Ubuntu/Debian que bem é tão semelhante ao Windows XP então tipo... Se você ao mesmo tempo não quer se matar no Terminal e só quer usar seu computador, fique a vontade pra usar o Mint ele é perfeito pra isso.

Informações principais:

Gerenciador de pacotes: APT (Advanced Package Tool).

Filosofia: Estabilidade de uso e experiência desktop agradável.

Modelo: Base Ubuntu LTS (No caso segue o modelo do Ubuntu).

Dificuldade: Baixa e moderada

Vantagens:

1) Sistema pronto para uso.

2) Interfave amigável.

3) Base Ubuntu/Debian o que garante estabilidade e suporte.

4) Ideal para desktops.

Desvantagens:

1) Não é rolling release, então os softwares podem vir de versões mais recentes.

2) Menos voltado para personalização profunda ou aprendizado avançado do Linux.

3) Foco em Desktop.

Site oficial do Linux Mint: https://linuxmint.com/
